# Contributing
Thanks for your interest in improving Deauther.

## Reporting issues
- Use the bug report template and include clear reproduction steps.
- Include your OS/distro, wireless adapter, and the exact command used.
- Attach relevant logs (redact sensitive data).

## Pull requests
- Keep changes focused and scoped to one topic.
- Follow existing shell style; keep scripts POSIX‑friendly where possible.
- Run `shellcheck deauther.sh` before submitting.

## Development notes
- This project targets Linux and relies on aircrack‑ng tooling.
